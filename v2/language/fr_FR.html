Introduction

Ce manuel décrit les fonctionnalités du plug-in MyParcel Belgique et vous explique comment l’installer dans votre boutique en ligne. Nous vous recommandons d’abord d’installer le plug-in dans un environnement de test avant de l’appliquer dans votre boutique web ‘live’. L’installation peut être réalisée de deux façons :

  1. via le répertoire de plug-in WordPress
  2. via un téléchargement manuel.

Sommaire

1 Installation

WDans tous les cas, nous recommandons d’abord de procéder à une installation dans un environnement de test et d’effectuer une sauvegarde de votre boutique en ligne.

1.1 Via le répertoire de plug-in WordPress

Pour installer le plug-in MyParcel via le répertoire du plug-in WordPress, procédez comme suit:

  1. Allez dans le menu Plug-ins de WordPress > Nouveau plugin.
  2. Tapez MyParcel et cliquez sur rechercher.
  3. Choisissez le plug-in WC MyParcel Belgique et cliquez sur ‘Installer maintenant’. Le plug-in va alors être téléchargé.
  4. Lorsque le téléchargement est terminé, cliquez sur Activer le plug-in’.

1.2 Téléchargement manuel

  1. Téléchargez le fichier zip du plug-in sur le site de WordPress: https://nl.wordpress.org/plugins/wc-myparcel-belgium/
  2. Dans WordPress, accédez au menu Plug-ins > Nouveau plug-in.
  3. Cliquez sur ‘Télécharger’ en haut de l'écran.
  4. Choisissez le fichier, puis cliquez sur ‘Installer maintenant’.
  5. Une fois l’installation terminée, cliquez sur ‘Activer le plug-in’.

1.3 Testen

Le plug-in MyParcel Belgique ajoute certaines options de livraison à votre checkout. Les options de livraison que vous souhaitez proposer à vos clients peuvent être définies dans la configuration du plug-in MyParcel. Nous vous conseillons donc de bien vérifier votre processus de commande après avoir installé le plug-in. Il est par exemple possible de paramétrer une livraison différée au niveau du checkout. Il est utile de pouvoir visualiser ce que vos clients pourront eux-mêmes voir.

Tous les checkouts dans Woocommerce ne fonctionnent pas de la même façon. Il est donc important de vérifier que tout est en ordre, que tout est bien correct et à la bonne place. Notre conseil est de contrôler chaque élément avant toute utilisation par vos clients.

Lorsque vous êtes connecté à votre compte MyParcel sur backoffice.sendmyparcel.be accédez au menuParamètres → Général.

2.3 Checkout

Dans l’onglet ‘Checkout’, vous avez la possibilité d’ajouter d’autres options d’expédition à votre checkout. Il est également possible de choisir de montrer le checkout non seulement au niveau des colis, mais aussi pour tous les modes d’expédition.

2.3.1 Options de livraison

Pour chacune des différentes options de livraison, vous pouvez paramétrer un supplément qui sera facturé à votre client via le checkout. Il s’agit donc de frais supplémentaires par rapport aux tarifs standards que vous avez déjà complétés. L’exemple ci-dessous vous montre comment ces suppléments sont repris dans le checkout. 

Au niveau de la configuration du checkout, 3 autres paramètres sont présents. Il s’agit des jours de livraison, de l’heure de clôture et du temps de traitement. Sur base de ces paramètres, basés sur votre méthode de traitement des commandes, les délais et jours de livraison corrects seront montrés à vos clients via le checkout.



2.3.2 Traitement des envois

Jours de livraison 

Vous indiquez ici les jours de dépôt de colis dans un point bpost. Vos clients se verront alors proposer des jours de livraison corrects et possibles.   

Heure de clôture

Vous pouvez préciser ici jusqu’à quelle heure les commandes peuvent être traitées. Si vous déposez les commandes dans un point bpost chaque jour vers 17:00, il vous sera plus facile de clôturer le traitement des commandes vers 16:00. Lorsque vos clients passent commande après cette heure, une date de livraison ultérieure leur sera montrée. Par exemple : l’heure de clôture est fixée à 16:00. Vous pouvez traiter les commandes le jour même et les déposer à temps. Votre client se verra alors proposer le lendemain avant 16:00 comme premier délai de livraison. Si le client commande après 16:00, la livraison sera pour le surlendemain.  

Temps de traitement

Vous pouvez ici indiquer le nombre de jours nécessaires au traitement d’une commande passée avant l’heure de clôture. Si vous indiquez ‘1 jour’, vous devrez tenir compte, dans l’affichage au niveau du checkout, que les colis ne seront pas livrés le lendemain. Par exemple : la commande est passée le 16 septembre, avant que vous ne vous rendiez au point bpost. Vous avez cependant configuré un temps de traitement d’un jour. Ce qui signifie que la première possibilité pour vous de déposer le colis est fixée au 17 septembre. Votre client se verra donc proposer le 18 septembre comme première date de livraison possible.  

3 Aperçu des commandes

L’aperçu des commandes vous permet de vérifier facilement le mode d’expédition relatif à la commande. Si vous le souhaitez, vous pouvez également modifier facilement le type de colis. Lorsque vous cliquez sur Colis, la fenêtre ci-dessous apparaît. Il est possible de désactiver les options d’expédition, ou bien d’ajouter ou de modifier le type d’envoi.

Après avoir choisi les options souhaitées ou modifié le Type d’envoi, cliquez sur ‘Enregistrer’. Vous pouvez ensuite exporter ce choix vers MyParcel BE et créer l’étiquette souhaitée. 



Vous pouvez transférer vos commandes vers MyParcel BE de deux manières, puis imprimer une étiquette. En sélectionnant une ou plusieurs commandes, puis en allant dans Actions. Vous pouvez ensuite faire votre choix : Exporter, Imprimer ou Exporter et Imprimer. Une icône de chargement est visible, tant en cas d’exportation par lot que d’exportation individuelle. La page est ensuite immédiatement rafraîchie (voir l’exemple ci-dessous). 



3.1 E-mail étiquette de retour

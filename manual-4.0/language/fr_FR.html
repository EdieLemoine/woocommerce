Introduction

Ce manuel décrit les fonctionnalités du plug-in MyParcel Belgique et vous explique comment l’installer dans votre boutique en ligne. Nous vous recommandons d’abord d’installer le plug-in dans un environnement de test avant de l’appliquer dans votre boutique web ‘live’. L’installation peut être réalisée de deux façons :

  1. via le répertoire de plug-in WordPress
  2. via un téléchargement manuel.

Sommaire

1 Installation

WDans tous les cas, nous recommandons d’abord de procéder à une installation dans un environnement de test et d’effectuer une sauvegarde de votre boutique en ligne.

1.1 Via le répertoire de plug-in WordPress

Pour installer le plug-in MyParcel via le répertoire du plug-in WordPress, procédez comme suit:

  1. Allez dans le menu Plug-ins de WordPress > Nouveau plugin.
  2. Tapez MyParcel et cliquez sur rechercher.
  3. Choisissez le plug-in WC MyParcel Belgique et cliquez sur ‘Installer maintenant’. Le plug-in va alors être téléchargé.
  4. Lorsque le téléchargement est terminé, cliquez sur Activer le plug-in’.

1.2 Téléchargement manuel

  1. Téléchargez le fichier zip du plug-in sur le site de WordPress: https://nl.wordpress.org/plugins/wc-myparcel-belgium/
  2. Dans WordPress, accédez au menu Plug-ins > Nouveau plug-in.
  3. Cliquez sur ‘Télécharger’ en haut de l'écran.
  4. Choisissez le fichier, puis cliquez sur ‘Installer maintenant’.
  5. Une fois l’installation terminée, cliquez sur ‘Activer le plug-in’.

1.3 Testen

Le plug-in MyParcel Belgique ajoute certaines options de livraison à votre checkout. Les options de livraison que vous souhaitez proposer à vos clients peuvent être définies dans la configuration du plug-in MyParcel. Nous vous conseillons donc de bien vérifier votre processus de commande après avoir installé le plug-in. Il est par exemple possible de paramétrer une livraison différée au niveau du checkout. Il est utile de pouvoir visualiser ce que vos clients pourront eux-mêmes voir.

Tous les checkouts dans Woocommerce ne fonctionnent pas de la même façon. Il est donc important de vérifier que tout est en ordre, que tout est bien correct et à la bonne place. Notre conseil est de contrôler chaque élément avant toute utilisation par vos clients.

2 Configuration

Allez dans le menu WooCommerce → MyParcel BE pour configurer le plug-in MyParcel.

2.1 Généralités

2.1.1 Paramètres API

Pour effectuer la connexion avec votre compte MyParcel, il est nécessaire de saisir votre clé API. 
Lorsque vous êtes connecté à votre compte MyParcel sur backoffice.sendmyparcel.be accédez au menuParamètres → Général.

2.1.2 Affichage des étiquettes

Indiquez ici si vous souhaitez télécharger immédiatement les étiquettes ou si vous souhaitez les ouvrir dans une nouvelle fenêtre.

2.1.3 Format de l’étiquette

Choisissez ici votre format d’étiquette, A6 ou A4.

2.1.4 Position de démarrage de l’impression

Cette option vous permet de redémarrer l’impression là où vous l’aviez laissée la dernière fois.

2.1.5 Track & Trace dans l’e-mail

Ajoutez le code Track & Trace aux e-mails envoyés aux clients. Attention ! Si vous choisissez cette option, assurez-vous que vous n’avez pas activé l’e-mail Track & Trace dans le back-office.

2.1.6 Track & Trace dans Ma Facture

Affichez le code et le lien Track & Trace dans Ma Facture.

2.1.7 Traitement direct des envois

Lorsque cette option est activée, les envois seront traités immédiatement dès leur transfert vers MyParcel.

2.1.8 Automatisation du statut des commandes

Le statut de la commande est modifié automatiquement après l’exportation réussie de celle-ci. Lorsque vous utilisez cette option en combinaison avec l’option Track & Trace dans l’e-mail, veillez à activer le traitement direct des envois. Si vous ne le faites pas, le code Track & Trace ne sera pas repris dans l’e-mail envoyé au client..

2.1.9 Conserver les anciens envois

Lorsque cette option est activée, les données des envois précédents (liens Track & Trace) seront conservées dans la commande si vous procédez plus d’une fois à une exportation.

2.1.10 Conserver le code-barres dans une note

Le code-barres est ajouté dans une note relative à la commande.

2.1.11 Outils de diagnostic

Vous pouvez activer les outils de diagnostic si un problème devait survenir au moment de l’exportation des envois. Ce fichier décrit la communication entre WooCommerce et MyParcel.

2.2 Paramètres d'exportation standards

2.2.1 Coupler des types de colis à des types d’expédition

Vous pouvez coupler des modes d’expédition à un type de colis pour que l’étiquette correcte soit créée dans votre compte MyParcel. Vous trouverez plus d’explications dans l’exemple ci-dessous. 



L’exemple montre que le mode d’expédition Tarif fixe est couplé au type de colis Colis.

2.2.2 Couplage de l’adresse e-mail du client

Vous pouvez choisir d'envoyer ou non l’e-mail de vos commandes à MyParcel BE. Lorsque vous associez l’adresse e-mail du client, vous pouvez configurer MyParcel BE pour que les e-mails Track & Trace soient envoyés à cette adresse. Vous pouvez activer ou désactiver cet e-mail dans votre back-end MyParcel BE et le formater selon votre charte graphique.

2.2.3 Couplage du numéro de téléphone du client

Lorsque vous associez le numéro de téléphone du client à un envoi, le livreur peut l’utiliser en vue de la distribution du colis. Cela permet d’augmenter considérablement les chances d’une livraison réussie des envois vers l’étranger.

2.2.4 Référence de l’étiquette

Cette option vous permet d’ajouter une référence à l’envoi. Celle-ci sera imprimée dans le coin supérieur gauche de l’étiquette et peut ensuite être utilisée pour rechercher un envoi dans l’aperçu MyParcel BE. Utilisez [ORDER_NR] pour indiquer le numéro de commande.

2.3 Paramètres du checkout

Dans l’onglet ‘Paramètres du checkout’, vous avez la possibilité d’ajouter d’autres options d’expédition à votre checkout. Il est également possible de choisir de montrer le checkout non seulement au niveau des colis, mais aussi pour tous les modes d’expédition.

2.3.1 Champs d’adresse MyParcel BE

Les champs d’adresse MyParcel BE sont utilisés lorsque cette option est activée. Cela signifie que la rue, le numéro de maison et l’extension sont présentés séparément. Si vous souhaitez utiliser les champs par défaut de WooCommerce, ne cochez pas cette case. 

2.3.2 Options de livraison MyParcel BE

Pour chacune des différentes options de livraison, vous pouvez paramétrer un supplément qui sera facturé à votre client via le checkout. Il s’agit donc de frais supplémentaires par rapport aux tarifs standards que vous avez déjà complétés. L’exemple ci-dessous vous montre comment ces suppléments sont repris dans le checkout.


2.3.3 Affichage

Vous pouvez associer des options de livraison à des méthodes d’expédition spécifiques en les ajoutant aux types de colis dans ‘Paramètres d’exportation standards’. Les options de livraison ne sont pas visibles pour les adresses étrangères.

2.3.4 Position des options de livraison

Vous pouvez ici modifier l’emplacement des options de livraison. Par défaut, celles-ci se trouvent sous les données de facturation.

2.3.5 Légendes

Vous pouvez placer une légende ‘options de livraison’ au-dessus des options MyParcel. S’il n’y a pas de légende, cet élément n’est pas visible. Lorsqu’aucune légende standard n’est complétée, les délais de livraison sont automatiquement affichés.

3 Bpost

Vous avez la possibilité d’activer ou de désactiver les paramètres de l’opérateur bpost via l’onglet ‘bpost’.



3.1 Paramètres d’exportation standards

Ces paramètres sont appliqués aux envois bpost que vous créez dans le back-end.

3.1.1 Envoi assuré (jusqu’à 500 €)

Avec cette option, les envois comprennent une assurance de manière standard. Par défaut, les envois ne sont pas assurés. Vous pouvez donc assurer vos envois si vous le souhaitez. Nous assurons la valeur d’achat de votre produit à concurrence de 500 €. Les colis assurés incluent toujours l’option ‘Signature pour réception’.

3.1.2 Signature pour réception

Avec cette option, les colis sont envoyés avec l’option ‘signature pour réception’ de manière standard. L’envoi est d’abord présenté à l’adresse personnelle. Si le destinataire est absent, le colis est alors livré chez les voisins. Dans les deux cas, une signature pour réception est requise.

3.2 Options de livraison bpost

En optant pour la livraison par bpost, vous activez également les options de livraison de cet opérateur.

3.2.1 Jours de livraison

Vous indiquez ici les jours de dépôt de vos colis dans un point bpost. Vos clients se verront alors proposer les jours de livraison corrects et possibles.

3.2.2 Heure de clôture

Vous pouvez préciser ici jusqu’à quelle heure les commandes peuvent être traitées. Si vous déposez les commandes dans un point bpost chaque jour vers 17:00, il vous sera plus facile de clôturer le traitement des commandes vers 16:00. Lorsque vos clients passent commande après cette heure, une date de livraison ultérieure leur sera montrée. Par exemple : l’heure de clôture est fixée à 16:00. Vous pouvez traiter les commandes le jour même et les déposer à temps. Avant 16:00, votre client se verra alors proposer le lendemain comme premier délai de livraison. Si le client commande après 16:00, la livraison sera pour le surlendemain.

3.2.3 Temps de traitement

Vous pouvez ici indiquer le nombre de jours nécessaires au traitement d’une commande passée avant l’heure de clôture. Si vous indiquez ‘1 jour’, vous devrez tenir compte, dans l’affichage au niveau du checkout, que les colis ne seront pas livrés le lendemain. Par exemple : la commande est passée le 16 septembre, avant que vous ne vous rendiez au point bpost. Vous avez cependant configuré un temps de traitement d’un jour. Ce qui signifie que la première possibilité pour vous de déposer le colis est fixée au 17 septembre. Votre client se verra donc proposer le 18 septembre comme première date de livraison possible.

3.2.4 Affichage de la date de livraison

La date de livraison est affichée dans les options de livraison.

3.2.5 Signature pour réception

Vous activez la possibilité pour vos clients de choisir l’option ‘Signature pour réception’ dans les options de livraison.

3.2.6 Coûts (optionnel)

Introduisez un montant positif ou négatif. Vous pouvez en effet proposer une réduction ou décompter des frais supplémentaires pour cette option de livraison.

3.3 Options de retrait bpost

3.3.1 Activation du retrait chez bpost

Vous activez ici les points de collecte bpost. Vos clients peuvent ainsi choisir de retirer leur colis dans un point bpost.

3.3.2 Coûts (optionnel)

Introduisez un montant positif ou négatif. Vous pouvez en effet proposer une réduction ou décompter des frais supplémentaires pour cette option de livraison.

5 Aperçu des commandes

L’aperçu des commandes vous permet de vérifier facilement le mode d’expédition relatif à la commande. Si vous le souhaitez, vous pouvez également modifier facilement le type de colis. Lorsque vous cliquez sur ‘Colis’, la fenêtre ci-dessous apparaît. Il est possible de désactiver les options d’expédition, ou bien d’ajouter ou de modifier le type d’envoi. Après avoir choisi les options souhaitées ou modifié le Type d’envoi, cliquez sur ‘Enregistrer’. Vous pouvez ensuite exporter ce choix vers MyParcel BE et créer l’étiquette souhaitée.  Vous pouvez transférer vos commandes vers MyParcel BE de deux manières, puis imprimer une étiquette. En sélectionnant une ou plusieurs commandes, puis en allant dans ‘Actions’. Vous pouvez ensuite faire votre choix : Exporter, Imprimer ou Exporter et Imprimer. Une icône de chargement est visible, tant en cas d’exportation par lot que d’exportation individuelle. La page est alors immédiatement rafraîchie (voir l’exemple ci-dessous). 



5.1 E-mail étiquette de retour

Vous pouvez envoyer un lien de retour à votre client depuis l’aperçu des commandes. Votre client reçoit un e-mail comprenant un lien lui permettant d’imprimer facilement une étiquette de retour. L’image ci-dessous vous explique quelle icône doit être utilisée pour ce faire.
